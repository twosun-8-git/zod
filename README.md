# react-hook-form

このリポジトリは**react-hook-form**の基本的な使い方を記載したリポジトリです。

## 🛠️ 開発環境

- Node.js（18.20.4）
- React（^18）
- Next（14.2.5）
- react-hook-form（^7.52.1）
- MUI（5.16.6）

## 📁 ディレクトリ構成

| ディレクトリ名 | 説明                                                               |
| -------------- | ------------------------------------------------------------------ |
| basic          | Zod の基本的な使い方                                               |
| sync           | メールアドレスのクロスフィールドバリデーションの発火タイミング修正 |

## 🛹 動作方法

#### 1. リポジトリをクローン

```
git clone https://github.com/twosun-8-git/zod.git
```

#### 2. モジュールをインストール

```
npm i
```

#### 3. 開発環境を起動

```
npm run dev
```

#### 4. http://localhost:3000 にアクセス
